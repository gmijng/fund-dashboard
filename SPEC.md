# 基金看盘工具 - 规格文档

## 概述
- 名称：基金看盘面板
- 类型：网页工具（本地 HTML + Node.js 后端）
- 功能：手动录入持仓，自动拉取实时行情，展示收益
- 目标用户：自己使用，不对外

## 页面布局

### 顶部：总览栏
- 总资产、总今日收益、总累计收益、持仓基金数
- 刷新时间

### 中部：指数栏
- 上证、深证、创业板、科创50、恒生、恒生科技

### 工具栏
- 搜索框（按名称/代码过滤）
- 添加基金 / 导入 / 清空数据
- 视图切换（简洁/详细）
- 导出 CSV
- 排序下拉
- 刷新按钮

### 底部：持仓表格

### 辅助功能
- 对比栏：选中多只基金对比涨跌
- 状态栏：错误信息提示

## 数据结构

```javascript
// 基金数据结构
{
  code: string,      // 基金代码
  name: string,      // 基金名称
  amount: number,    // 持有金额（元）- 用户手动填写
  gain: number,      // 持有收益（元）- 用户手动填写
  cost: number       // 持仓成本（当前市值）
}
```

**计算逻辑：**
- 累计收益率 = 持有收益 / 持有金额 × 100%
- 今日收益 = 份额 × (今日估值 - 昨日净值)
- 份额 = 持有金额 / 今日估值（估算）

## 数据来源
- 基金实时估值：`https://fundgz.1234567.com.cn/js/{code}.js`
- A股指数：`https://qt.gtimg.cn/q=sh000001,...`
- CORS代理：`https://api.codetabs.com/v1/proxy`

## 技术方案
- 前端：原生 HTML + CSS + JavaScript（无框架）
- 数据存储：LocalStorage 存持仓配置（JSON）
- 刷新：每 30 秒自动刷新，或手动点击

## 交互
- **添加基金**：输入代码 → 自动获取名称 → 填入持有金额和收益 → 确认
- **编辑**：直接点击持有金额/收益输入框修改
- **删除**：点击 × 按钮
- **搜索**：输入名称或代码实时过滤
- **排序**：点击表头排序，或使用下拉框
- **对比**：点击 + 添加基金到对比栏，最多4只
- **导出**：生成 CSV 文件
- **视图切换**：简洁模式隐藏部分列

## 视觉样式
- 深色主题（暗色背景，深色卡片）
- 涨红跌绿
- 简洁专业，类似专业交易终端风格
- 响应式：电脑/手机都能看

### 状态高亮
- 可疑基金（收益率 ±1% 且金额 >5000）：黄色背景
- 高盈利（收益率 >10%）：红色背景
- 高亏损（收益率 <-10%）：绿色背景

## 快捷键
- `R`：刷新数据
