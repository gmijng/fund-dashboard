# 基金看盘工具 - 规格文档

## 概述
- 名称：基金看盘面板
- 类型：网页工具（本地 HTML + Node.js 后端）
- 功能：手动录入持仓，自动拉取实时行情，展示收益
- 目标用户：自己使用，不对外

## 页面布局

### 顶部：总览栏
- 总资产、总今日收益、总累计收益
- 刷新时间

### 中部：持仓表格
| 列 | 说明 |
|---|---|
| 基金名称/代码 | 手动输入 |
| 持有份额 | 手动输入 |
| 成本价 | 手动输入 |
| 持仓金额 | =份额×成本价 |
| 今日估值 | 自动抓取（fundgz API） |
| 今日涨跌% | 自动计算 |
| 持仓收益 | 自动计算 |
| 累计收益% | 自动计算 |

### 底部：板块行情（可选）
- 科创50、恒生科技、创业板等主要指数涨跌

## 数据来源
- 基金实时估值：`https://fundgz.1234567.com.cn/js/{code}.js`
- A股指数：`https://hq.sinajs.cn/list=sh000001,...`
- 港股：`https://hq.sinajs.cn/list=hkHSI,hkHSTECH`

## 技术方案
- 前端：原生 HTML + CSS + JavaScript（无框架）
- 后端：Node.js（轻量，用 express 或原生 http）
- 数据存储：LocalStorage 存持仓配置（JSON）
- 刷新：每 30 秒自动刷新，或手动点击

## 交互
- 添加基金：输入代码 → 自动获取名称 → 填入份额和成本
- 删除基金：点击删除按钮
- 编辑：直接修改表格内容
- 数据持久化：刷新页面不丢失数据

## 样式
- 深色主题（暗色背景，深色卡片）
- 涨红跌绿
- 简洁专业，类似专业交易终端风格
- 响应式：电脑/手机都能看

## 交付物
- 一个可运行的 HTML 文件 + Node.js server（可合并）
- 双击打开即用，或 `node server.js` 启动
